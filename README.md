# IVT RFID Tracking System

Static mock UI prototype for an IVT factory RFID tracking system.

## Screens

- Dashboard
- Live Tracking
- Asset Registry
- Scan Events
- Alerts
- Reports
- RFID Readers
- Settings

## Deploy

This project is plain HTML, CSS, and JavaScript. It can be hosted for free on GitHub Pages.

Recommended GitHub Pages settings:

- Source: Deploy from a branch
- Branch: main
- Folder: /root
